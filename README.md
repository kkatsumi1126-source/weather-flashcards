# 気象フラッシュカード

気象学習用の静的フラッシュカード集です。Macではローカルで、スマホではGitHub Pagesで公開したページをブラウザから開いて使えます。

すべて静的ファイルだけで動く構成です。トップ画面の `index.html` から各教材を開けます。

## 構成

- `index.html`: トップ画面
- `国際式天気記号フラッシュカード/`: 現在天気 `ww` のカード
- `台風・熱帯低気圧フラッシュカード/`: 台風・熱帯低気圧のカード
- `海上警報フラッシュカード/`: 海上警報のカード
- `気象衛星雲パターンフラッシュカード/`: 衛星雲パターンのカード
- `*.command`: Macでローカル起動するためのファイル。GitHub Pagesでは使いませんが、残してあります。

## Macでの起動方法

Macでは、次のどちらかで起動できます。

- `トップ画面を開く.command` をダブルクリックする
- `index.html` をブラウザで開く

各教材フォルダ内の `起動.command` をダブルクリックして、教材を直接開くこともできます。

## スマホで使う方法

スマホでは、GitHub Pagesで公開されたURLをブラウザで開きます。アプリのインストールは不要です。

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成します。
2. この「気象フラッシュカード」フォルダの中身を、リポジトリのルートにアップロードします。
3. GitHubのリポジトリ画面で `Settings` を開きます。
4. `Pages` を開きます。
5. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
6. `Branch` で `main`、フォルダで `/ (root)` を選んで保存します。
7. 数分待つと、GitHub PagesのURLが表示されます。
8. 表示されたURLをスマホのブラウザで開くと使えます。

## 注意

- すべて静的ファイルだけで動きます。
- CSS、JavaScript、データ、画像は相対パスで読み込む構成です。
- 学習記録はブラウザの保存領域に保存されます。端末やブラウザを変えると記録は共有されません。

## 天気記号画像の追加

国際式天気記号の画像を追加する場合は、次のフォルダに入れます。

`国際式天気記号フラッシュカード/symbols/`

画像ファイル名の例:

`国際式天気記号フラッシュカード/symbols/ww_61.png`

カードデータの `image` に指定された画像がある場合は、その画像を表示します。画像がない場合は、表面に `ww 61` のような番号を表示します。
