# 気象衛星雲パターンフラッシュカード

このアプリは気象予報士試験の学習用フラッシュカードです。
気象衛星画像で見る特徴的な雲の形・雲パターンの名前、意味、見たら考えることを学習できます。

## 使い方

- `index.html` をダブルクリックすると使えます。
- Macでは `起動.command` をダブルクリックしても開けます。
- インターネット接続なしで使えます。

## 学習記録

- 学習状況はブラウザ内に保存されます。
- 前回選んだフィルター、最後に使ったカード番号、覚えた枚数、もう一度の枚数が保存されます。
- 「学習記録をリセット」を押すと保存データが消えます。
