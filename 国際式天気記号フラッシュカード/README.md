# 国際式天気記号フラッシュカード

国際式天気記号の現在天気 `ww` を覚えるためのフラッシュカードです。

- `present_weather_major_cards.js` にカードデータを置いています。
- `index.html` を開くと学習できます。
- `起動.command` をダブルクリックしても開けます。
- GitHub Pagesでは、このフォルダを含むリポジトリを公開すればそのまま使えます。
- 記号画像を追加する場合は、`symbols/ww_61.png` のような相対パスで配置してください。
- 画像がないカードは、表面に `ww 61` のような番号を表示します。
